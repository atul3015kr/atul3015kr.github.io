My Projects

1) Save-The-Ball

...A double player 2D tennis game • June 2017...

This is a simple 2D tennis game which can be played between two players.
Each player has been assigned a couple of keys to control the racquet. To provide
players a challenging game, the speed of the ball will keep increasing after each complete
cycle, i.e, each time the ball hits racquet of player 1.
I used Java graphic library Swing and introduced the concept of threading to build this game.
